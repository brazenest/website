# Legacy Next.js v2 Reference

This directory preserves the original Next.js implementation as migration reference material.

## Boundary Note

- The active v3 app lives at the repository root and builds from `src/`.
- `next-app/` should not be wired into active v3 imports, aliases, or build configuration.
- Any temporary lookups into this directory should stay clearly marked as reference-only migration work.
